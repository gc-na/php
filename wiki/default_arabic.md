# القيمة الافتراضية في PHP: دليل شامل للمطورين

## ملخص
في PHP، تُستخدم الكلمة الرئيسية "default" لتحديد القيم الافتراضية في هياكل التحكم مثل جمل switch. تعتبر هذه الميزة مفيدة لتحديد سلوك البرنامج عندما لا تتطابق أي من الحالات المحددة.

## الوثائق
تستخدم "default" بشكل رئيسي في جمل switch، حيث تُحدد ما يجب أن يحدث إذا لم تتطابق قيمة المتغير مع أي من الحالات المذكورة. يمكن أن تتواجد "default" في نهاية جملة switch، ويجب أن تكون خارج الأقواس الخاصة بالحالات. هذه الميزة تساعد على تحسين قراءة الكود وتسهيل صيانته.

### الاستخدام
```php
switch ($variable) {
    case 'value1':
        // تنفيذ الكود إذا كانت القيمة 'value1'
        break;
    case 'value2':
        // تنفيذ الكود إذا كانت القيمة 'value2'
        break;
    default:
        // تنفيذ الكود إذا لم تتطابق أي من القيم
        break;
}
```

## الأمثلة
### مثال 1: استخدام default في switch
```php
$day = "الإثنين";
switch ($day) {
    case "الأحد":
        echo "اليوم هو الأحد.";
        break;
    case "الإثنين":
        echo "اليوم هو الإثنين.";
        break;
    default:
        echo "اليوم غير معروف.";
        break;
}
```

### مثال 2: استخدام default مع قيم متعددة
```php
$fruit = "تفاح";
switch ($fruit) {
    case "موز":
        echo "فواكه استوائية.";
        break;
    case "تفاح":
    case "كمثرى":
        echo "فواكه شتوية.";
        break;
    default:
        echo "فواكه غير معروفة.";
        break;
}
```

## الشرح
من الشائع أن يخطئ المبرمجون في وضع جملة "default" في مكان غير صحيح، مثل داخل حالة معينة. يجب أن تكون "default" في نهاية جملة switch. أيضًا، إذا كانت هناك حاجة للتأكد من تنفيذ بعض الشيفرات حتى في حالة عدم تطابق أي من الحالات، يجب استخدام "default" بشكل صحيح.

## ملخص في جملة واحدة
تُستخدم "default" في PHP لتحديد سلوك البرنامج عند عدم تطابق القيم في جملة switch، مما يعزز تنظيم الكود ووضوحه.
