# PHP中的elseif語句：用法與示例

## 概述
`elseif` 是 PHP 中的一個控制結構，用於在多個條件判斷中提供額外的判斷選項。它通常與 `if` 語句配合使用，允許根據不同的條件執行不同的代碼塊。

## 文檔
### 目的
`elseif` 是用來處理多條件判斷的語句，當第一個 `if` 條件不成立時，程序會檢查後續的 `elseif` 條件，直到找到一個為真的條件或所有條件都檢查完畢。

### 語法
```php
if (condition1) {
    // 當 condition1 為真時執行的代碼
} elseif (condition2) {
    // 當 condition2 為真時執行的代碼
} else {
    // 當以上條件都不成立時執行的代碼
}
```

### 使用方法
- 使用 `if` 開始條件判斷。
- 在需要的地方使用 `elseif` 來添加更多的條件。
- 最後可以選擇性使用 `else` 來處理所有條件都不成立的情況。

## 示例
### 基本用法
```php
$score = 75;

if ($score >= 90) {
    echo "成績優秀";
} elseif ($score >= 75) {
    echo "成績良好";
} elseif ($score >= 60) {
    echo "成績及格";
} else {
    echo "成績不及格";
}
```
在這個例子中，根據 `$score` 的值，將輸出對應的成績評價。

## 解釋
### 常見陷阱
- **括號錯誤**：確保每個條件都正確包裹在括號中，否則會導致語法錯誤或邏輯錯誤。
- **未使用大括號**：雖然在只有一行代碼時可以省略大括號，但建議始終使用，以提高代碼可讀性並避免潛在錯誤。

### 額外說明
- `elseif` 可以有多個，根據需要可以連續添加。
- 可以在 `elseif` 和 `else` 之間添加其他控制結構，如 `switch`，但要注意代碼的可讀性。

## 一行總結
`elseif` 是 PHP 中用於多條件判斷的關鍵字，便於根據不同條件執行不同的代碼。
