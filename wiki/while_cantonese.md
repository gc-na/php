# PHP 的 while 迴圈：用法與示例

## 概述
`while` 是 PHP 中的一個控制結構，用於在滿足特定條件時重複執行一段代碼。它在需要根據條件動態執行代碼時非常有用，能夠使程式更加靈活和高效。

## 文檔
### 目的
`while` 迴圈的主要目的是在條件為真時持續執行代碼塊。這使得開發者能夠根據變數的狀態進行重複操作，直到條件不再滿足。

### 用法
`while` 的基本語法結構如下：

```php
while (條件) {
    // 執行的代碼
}
```
- **條件**：這是一個布爾表達式，當其值為 `true` 時，迴圈將繼續執行；當其值為 `false` 時，迴圈將停止。
- **執行的代碼**：這是當條件為真時所執行的代碼塊。

### 詳細說明
- `while` 迴圈的執行會在每次迴圈開始時評估條件。如果條件為真，則執行代碼塊；否則，跳出迴圈。
- 需要注意的是，若條件永遠為真，則會導致無限迴圈，這會使程式無法正常運行。

## 示例
### 基本用法示例
以下是一個簡單的 `while` 迴圈示例，該示例從 1 數到 5：

```php
$count = 1;

while ($count <= 5) {
    echo $count;
    $count++;
}
```
在這個例子中，程式會輸出 `12345`，因為在每次迴圈中，變數 `$count` 會遞增，直到它的值超過 5。

## 解釋
### 常見陷阱
- **無限迴圈**：如果忘記在迴圈內部更新條件變數，則可能會導致無限迴圈。例如：

```php
$count = 1;

while ($count <= 5) {
    echo $count; // 忘記 $count++ 將導致無限迴圈
}
```
- **條件評估**：確保條件能夠在某個時點變為 `false`，以避免程序卡住。
- **代碼塊的範圍**：注意 `{}` 中的代碼塊範圍，確保所有需要重複的代碼都包含在內。

## 一句總結
`while` 迴圈是 PHP 中一個強大的工具，能夠根據特定條件重複執行代碼塊，適合用於需要動態控制重複操作的情況。
