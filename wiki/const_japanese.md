# PHPにおける「const」の使用法と詳細

## 概要
PHPの「const」は、クラス内で定義される定数を宣言するためのキーワードです。定数は、変更できない値を保持し、コードの可読性と保守性を向上させます。

## ドキュメント
### 目的
`const`は、クラス内で定数を定義するために使用されます。この定数は、クラスがインスタンス化される前から利用可能であり、他のクラスやメソッドからもアクセスできます。

### 使用法
- **定義**: `const`キーワードを使用して定数を定義します。
- **アクセス**: 定数には、`self::CONSTANT_NAME`または`ClassName::CONSTANT_NAME`の形式でアクセスします。

### 詳細
- **スコープ**: 定数は、定義されたクラス内でのみスコープが有効です。外部からはアクセスできません。
- **型**: 定数は、値の型を持ちませんが、整数、浮動小数点数、文字列、配列を含むことができます。
- **定数の再定義**: 一度定義された定数は再定義できません。

## 例
### 基本的な使用例
```php
class MyClass {
    const MY_CONSTANT = '定数の値';

    public function showConstant() {
        echo self::MY_CONSTANT;
    }
}

$obj = new MyClass();
$obj->showConstant(); // 出力: 定数の値
```

### 数値定数の例
```php
class Math {
    const PI = 3.14;

    public function getPi() {
        return self::PI;
    }
}

$math = new Math();
echo $math->getPi(); // 出力: 3.14
```

## 説明
### 注意点
- **変更不可**: `const`で定義した定数は変更できないため、コード内で再利用することが推奨されます。
- **アクセス修飾子**: `const`は、公開（public）アクセス修飾子を持ち、クラス外からもアクセス可能です。
- **バージョン依存**: PHP 5.3以降で使用できますが、古いバージョンではサポートされていません。

### よくあるエラー
- 定数を再定義しようとすると、エラーが発生します。
- クラス外から定数にアクセスする際には、正しいクラス名を使用する必要があります。

## 一文要約
PHPの`const`は、クラス内で不変の定数を定義するためのキーワードであり、コードの明確性を向上させます。
